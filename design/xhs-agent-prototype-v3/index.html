Design Note

市面上 99% 的 AI 创作工具都长成"又一个紫色 SaaS Dashboard"。这不是。

这一版把产品当成编辑工作室来组织：每篇笔记是一份稿件、每次诊断是一次审稿会、每个工具调用是一次编辑动作。

米白纸张、墨色文字、衬线 display、单点小红书红 accent——克制而不冷淡，密集而不嘈杂。

A

B
增长专家

C
视觉专家
纯文字封面在穿搭赛道几乎没有竞争力。必须有真人出镜或平铺图。建议用 AI 生成一张「奶茶色叠穿」的场景封面。
综合裁判结论
可发布，但建议先补封面图 + 价格锚点再发。预估优化后互动率提升 25-35%。

07模板库

选一个结构模板，给一个主题，AI 按骨架生成完整笔记
踩坑避雷 经验分享
先说结论 → 踩坑经历 → 正确做法 → 总结建议。适合护肤/装修/购物类。
## 结论先行 {一句话结论} ## 我的踩坑经历 {具体故事} ## 正确做法 {步骤清单} ## 总结 {一句话收尾 + CTA}
清单合集 种草推荐
开头定调 → N 个推荐项（名称+理由+价格）→ 总结。适合好物/书单/歌单。
## 开头 {场景+痛点} ## 推荐清单 1. {名称} · {价格} {一句话理由} 2. ... ## 总结 {使用场景建议}
种草测评 产品评测
产品信息 → 使用体验 → 优缺点 → 适合人群 → 购买建议。
## 产品信息 {品牌/名称/价格/规格} ## 使用体验 {真实感受} ## 优缺点 ✅ ... ❌ ... ## 推荐指数 {评分 + 适合人群}
情感共鸣 故事向
情绪钩子 → 故事展开 → 转折/感悟 → 金句收尾。适合成长/情感/职场。
## 钩子 {引发共鸣的一句话} ## 故事 {具体经历} ## 感悟 {转折 + 成长} ## 金句 {可被截图的一句话}
教程攻略 干货教学
效果展示 → 准备材料 → 分步教程 → 注意事项 → 成果对比。
## 效果预览 {成品展示} ## 你需要 {材料/工具清单} ## 步骤 1. {配图说明} 2. ... ## 避坑提醒 {常见错误}
