# 小红书 Agent · v2 原型页面与功能清单

| 模块 | 功能覆盖 |
|---|---|
| 登录注册 | 登录、注册、鉴权提示 |
| 创作对话 | 历史侧栏、当前笔记上下文、图片上传、工具调用、后台续跑 |
| 笔记列表 | 搜索、筛选、状态、评分、图片数、管理员按用户分组 |
| 笔记编辑 | 三栏工作台、标题正文标签、图片队列、首图即封面、版本、违禁词、发布 |
| 发布诊断 | 历史诊断、重新诊断、应用优化方案、五维评分、专家建议 |
| 模板库 | 创建、删除、提取模板、套用主题 |
| 数据页 | 总量、发布状态、热门标签、日历 |
| 任务中心 | 后台任务、耗时、trace、取消、恢复 |
| 设置页 | 模型队列、独立 URL/key、支持 URL/quality、静态图片公网测试、真实调用测试、改密码、用户管理 |
| 全局组件 | 命令面板、新手引导、图片编辑器 |

## v2 设计要求
- 少说明，多产品现场。
- 右侧 Live Context 常驻。
- 动效是状态反馈，不是装饰。
- 所有图片均为诚实占位，不伪造真实素材。
